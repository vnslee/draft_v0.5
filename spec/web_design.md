# 웹 디자인 가이드

## 공통사항

### Reference

### 화면 체계
- 메인화면 (M1)
  - 인터랙티브 지도가 꽉차도록 구성(별도 팝업이 화면에 노출된 상태일 땐 반투명, 그 외에는 지도와 국가/권역이 잘 구분될 수 있도록 선명하게 구성)
  - 최상단에 현대캐피탈 CI로고와 "Hyundai Capital" 타이틀 중앙 정렬로 표시
  - 최상단 우측에는 한/영 번역 버튼, 톱니바퀴 모양의 설정 버튼만 존재
  - 화면 정중앙에는 챗봇 팝업(C1) Default 노출
  - 챗봇 위젯 버튼은 화면 좌측 하단에 적절한 버튼으로 구성
  - 진출국가, 진출예정국가 마크로 지도에 표시
  - 지도 최우측 상단에 범례표시로 진출국가, 진출예정국가 분류 필요
- 챗봇 팝업 (C1)
  - 메인화면 좌우 사이드 50%, 상하 50%를 제외한 중간 사이즈의 팝업
  - 정중앙에는 대화창과 답변창 구성
  - 대화창과 답변창 사이에는 Mode (내부 자료 기반 답변 / 외부 리서치 기반 답변) 선택 콤보 또는 버튼 추가
- 국가 요약정보 팝업 (P1)
  - 메인화면 좌우 사이드 50%, 상하 50%를 제외한 중간 사이즈의 팝업
  - 진출여부, 시장, 핵심규제, 특화요건 항목 노출
  - 화면 최우측 하단 상세보기, 보고서, 보고서 생성 버튼 구성
- 국가 상세정보 팝업 (P2)
  - 메인화면 좌우 사이드 50%, 상하 50%를 제외한 중간 사이즈의 팝업
  - 화면 상단
    - 국기, 국가명, 국가영문명 노출
    - 최우측 시뮬레이션 버튼 구성
  - 화면 중간 영역
    - 국가일반, 시스템 정보 등을 표출해야함.
    - 표, 차트 등 적절한 데이터 시각화 필요
  - 화면 최우측 하단 보고서, 보고서 생성 버튼 구성
- 권역 요약정보 팝업 (P3)
  - 메인화면 좌우 사이드 50%, 상하 50%를 제외한 중간 사이즈의 팝업
  - 기진출 국가 리스트 표시
  - 진출 예정 국가에 대한 quick-win scoring 순위, it준비도, biz난이도, 종합점수 노출
  - 화면 최우측 하단 상세보기, 보고서, 보고서 생성 버튼 구성
- 권역 상세정보 팝업 (P4)
  - 메인화면 좌우 사이드 50%, 상하 50%를 제외한 중간 사이즈의 팝업
  - 화면 상단
    - 권역명 노출
    - 최우측 [시뮬레이션], [보고서] 버튼 구성
  - 화면 중간 영역
    - 권역 기본 정보 노출
    - 표, 차트 등 적절한 데이터 시각화 필요
  - 화면 최우측 하단 보고서, 보고서 생성 버튼 구성
- 국가 진단 보고서 팝업 (PR1)
  - 메인화면 좌우 사이드 20%, 상하 20%를 제외한 풀사이즈 팝업
  - 최상단 국기, 국가명, 국가영문명 노출
    - 해당 정보 하단에 권역과 비교국 국가명 노출
  - 최우측 상단 보고서 id, 보고서 생성일시, 비교국 스냅샷 일자 정보 노출
  - 최우측 상단 보고서 pdf 다운로드 버튼 구성
  - 중간 영역 내 탭으로 구분하여 구성
    - 요약, 판정근거, 비용 추정, 리스크, 가중치, 출처
    - 각 탭 영역별로 적절한 레이아웃 구성 필요
- 권역 진단 보고서 팝업 (PR2)
  - 메인화면 좌우 사이드 20%, 상하 20%를 제외한 풀사이즈 팝업
  - 최상단 지구본, 권역명, 권역영문명 노출 (ex. 유럽(Europe) 권역 Quick-Win 분석 보고서)
  - 최우측 상단 보고서 id, 보고서 생성일시
  - 최우측 상단 보고서 pdf 다운로드 버튼 구성
  - 중간 영역 내 탭으로 구분하여 구성
    - 요약, 시장, 규제, 상품, 시스템
    - 각 탭 영역별로 적절한 레이아웃 구성 필요
- 룰셋 설정 팝업 (PS1)
  - 메인화면 좌우 사이드 20%, 상하 20%를 제외한 풀사이즈 팝업
  - 화면 우측 상단에 가중치 룰셋 ID 드롭다운
  - 화면 우측 하단에 [저장] 버튼
  - 총 3개의 패널로 구성
    - 카테고리 가중치
      - 총 합계 100% 로 표현되는 슬라이더(왼쪽 0, 오른쪽 100%)
        - 시장
        - 규제
        - 환경(금융)
        - 시스템
      - 임계값 신뢰도 계수: 각 계수는 0부터 100까지
        - 이식 임계
        - 시스템 게이트
      - 출처 신뢰도 계수: 0부터 1.0 까지
        - Tier 1(공식)
        - Tier 2(준공식)
        - Tier 3(참고)
- 프로그레스 팝업 (PS2)
  - 총 5개의 프로그레스 바가 병렬 구성 (결과 생성 영역은 상위 4개 영역과 분리 필요)
    - 시장
    - 규제
    - 상품
    - 시스템
    - 결과 생성
  - 우측 하단에 [보고서 보기] 버튼 구성
  
### 화면 구성 시 공통사항
- 사이드메뉴 구성 없음
- 팝업 화면 상단 우측에 닫기 버튼 구성
- 만약 프로그레스 팝업(PS2)가 화면에 활성화된 상태가 아니고 보고서 생성 진행 중인건이 있다면 화면 우측 상단에 프로그레스를 확인할 수 있는 별도 카드 형태의 프로그레스바 노출
  - 해당 프로그레스 패널 내에는 국가/권역 단위 정보와 전체 진행률 0-100% 표시, 상세보기 버튼 구성 (클릭 시 해당 패널 축소 + 프로그레스 팝업 (PS2) 화면 정중앙 노출)
- 화면 정중앙에 챗봇 팝업(C1)과 프로그레스 팝업(PS2)을 제외한 별도 팝업이 노출되있는 경우 챗봇 팝업(C1)은 항상 화면 좌측 하단에 위치
- 화면 정중앙에 챗봇 팝업(C1)과 프로그레스 팝업(PS2)을 제외한 별도 팝업이 노출되어있지 않은 경우에는 챗봇 팝업(C1)은 항상 화면 정중앙에 위치
- 보고서 생성 진행중이지 않다면 프로그레스 패널과 프로그레스 팝업(PS2)은 항상 노출하지 않음.

### 화면 흐름 예시
1. 메인화면(M1)진입
2. 챗봇 팝업 화면(C1) 정중앙 노출
3. 챗봇에서 제한된 질문 리스트 중 선택 또는 사용자가 직접 입력
4. 챗봇에서 선택 또는 입력한 내용에 따라 아래와 같이 분기
   4-1. 특정 국가명 INPUT -> 챗봇 팝업(C1)이 메인화면(M1) 좌측 하단 영역으로 이동(애니메이션 적용) -> 해당 국가 권역 지점으로 지도 확대 (애니메이션 적용을 통해 점진적 확대) -> 해당 국가 마커 표시 (깜빡이는 형태의 에니메이션 적용)  
        -> 사용자가 입력한 INPUT에 따라서 챗봇 내 제한된 질문 리스트 선택  
           -> "국가 요약정보 확인" 클릭 시 -> 국가 요약정보 팝업(P1) 화면 정중앙 노출 (해당 국가 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용) 
                                          -> 국가 요약정보 팝업(P1) 내 상세보기 버튼 클릭 시 -> 국가 요약 정보 팝업(P1) 숨김 처리 -> 국가 상세정보 팝업(P2) 노출 
                                          -> 국가 요약정보 팝업(P1) 내 보고서 버튼 클릭 시 -> 국가 요약 정보 팝업(P1) 숨김 처리 -> 국가 진단 보고서 팝업(PR1) 노출
                                          -> 국가 요약정보 팝업(P1) 내 보고서 생성 버튼 클릭 시 -> 국가 요약 정보 팝업(P1) 숨김 처리 -> 프로그레스 팝업 (PS2) 노출
           -> "국가 상세정보 확인" 클릭 시 -> 국가 상세정보 팝업(P2) 화면 정중앙 노출 (해당 국가 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용)
                                          -> 국가 상세정보 팝업(P2) 내 보고서 버튼 클릭 시 -> 국가 상세정보 팝업(P2) 숨김 처리 -> 국가 진단 보고서 팝업(PR1) 노출
                                          -> 국가 상세정보 팝업(P2) 내 보고서 생성 버튼 클릭 시 -> 국가 상세정보 팝업(P2) 숨김 처리 -> 프로그레스 팝업(PS2) 노출
           -> "국가 진단 보고서 보기" 클릭 시 -> 국가 진단 보고서 팝업(PR1) 화면 정중앙 노출 (해당 국가 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용)
                                            -> 국가 진단 보고서 팝업(PR1) 내 보고서 출력 버튼 클릭 시 -> 국가 진단 보고서 팝업(PR1) 숨김 처리 -> 프로그레스 팝업(PS2) 노출
                                            -> 국가 진단 보고서 팝업(PR1) 내 룰셋 설정 버튼 클릭 시 -> 국가 진단 보고서 팝업(PR1) 숨김 처리 -> 룰셋 설정 팝업(PS1) 노출
   4-2. 특정 권역 INPUT -> 챗봇 팝업(C1)이 메인화면(M1) 좌측 하단 영역으로 이동(애니메이션 적용) -> 해당 권역 지점으로 지도 확대 (애니메이션 적용을 통해 점진적 확대) -> 해당 권역 라인 또는 범위 표시 (깜빡이는 형태의 에니메이션 적용)  
        -> 사용자가 입력한 INPUT에 따라서 챗봇 내 제한된 질문 리스트 선택  
           -> "권역 요약정보 확인" 클릭 시 -> 권역 요약정보 팝업(P3) 화면 정중앙 노출 (해당 권역 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용) 
                                          -> 권역 요약정보 팝업(P3) 내 상세보기 버튼 클릭 시 -> 권역 요약정보 팝업(P3) 숨김 처리 -> 권역 상세정보 팝업(P4) 노출 
                                          -> 권역 요약정보 팝업(P3) 내 보고서 버튼 클릭 시 -> 권역 요약정보 팝업(P3) 숨김 처리 -> 권역 진단 보고서 팝업(PR2) 노출
                                          -> 권역 요약정보 팝업(P3) 내 보고서 생성 버튼 클릭 시 -> 권역 요약정보 팝업(P3) 숨김 처리 -> 프로그레스 팝업 (PS2) 노출
           -> "권역 상세정보 확인" 클릭 시 -> 권역 상세정보 팝업(P4) 화면 정중앙 노출 (해당 권역 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용)
                                          -> 권역 상세정보 팝업(P4) 내 보고서 버튼 클릭 시 -> 권역 상세정보 팝업(P4) 숨김 처리 -> 권역 진단 보고서 팝업(PR2) 노출
                                          -> 권역 상세정보 팝업(P4) 내 보고서 생성 버튼 클릭 시 -> 권역 상세정보 팝업(P4) 숨김 처리 -> 프로그레스 팝업 (PS2) 노출
           -> "권역 진단 보고서 보기" 클릭 시 -> 권역 진단 보고서 팝업(PR2) 화면 정중앙 노출 (해당 권역 마커에서 팝업이 튀어나오는 듯한 애니메이션 적용)
                                            -> 권역 진단 보고서 팝업(PR2) 내 보고서 출력 버튼 클릭 시 -> 권역 진단 보고서 팝업(PR2) 숨김 처리 -> 프로그레스 팝업(PS2) 노출
                                            -> 권역 진단 보고서 팝업(PR2) 내 룰셋 설정 버튼 클릭 시 -> 권역 진단 보고서 팝업(PR2) 숨김 처리 -> 룰셋 설정 팝업(PS1) 노출
   4-3. 특정 국가에 대한 질문 시
