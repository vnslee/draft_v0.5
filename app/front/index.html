📈 스코어링 엔진

Python 기반 동적 스코어링 엔진이 국가별 데이터를 분석하여 시장 진출 적합성을 평가합니다.

🎨 HTML 렌더링

Jinja2 템플릿 엔진으로 JSON 보고서를 전문적인 HTML 보고서로 동적 변환합니다.

자세히 보기

🔌 REST API

FastAPI 백엔드가 데이터 및 보고서 조회, 렌더링 등의 기능을 제공합니다.

API 문서
🎯 스코어링 엔진

scoring_engine.py

기능: 국가별 데이터를 받아 활성 항목(active_items)만으로 동적 계산

  • ✓ 비즈니스 점수 (매력도 & 난이도)
  • ✓ IT 준비도 점수 (유사도 기반)
  • ✓ 비용 추정 (할인율 적용)
  • ✓ 게이트 체크 (규제 준수)
  • ✓ 실사 우선순위 (Tier 기반)
python3 scoring_engine.py PL "추가항목"
🎨 동적 HTML 렌더링

report_renderer.py & report_template.html

기능: JSON 보고서를 Jinja2 템플릿으로 전문적인 HTML로 변환

  • ✓ 6개 탭 구조 (요약, 판정근거, 비용, 리스크, 가중치, 출처)
  • ✓ Kinetic Enterprise 디자인 시스템
  • ✓ 반응형 레이아웃 (모바일/태블릿/데스크톱)
  • ✓ PDF 인쇄 최적화
  • ✓ 실시간 데이터 바인딩
python3 report_renderer.py report/country/PL/PL_rpt_2026-06-18T1500.json

보고서 목록 로드 중...

    🔌 API 엔드포인트

    데이터 API

    GET /api/data/country/{CODE} 국가 데이터 조회
    GET /api/data/internal 가중치 & 규칙 조회

    보고서 API

    GET /api/report/country/{CODE} 보고서 HTML 조회
    GET /api/report/country/{CODE}/json 보고서 JSON 조회
    POST /api/render/report/{CODE} 보고서 렌더링 요청
    GET /api/report/available-countries 사용 가능한 국가 목록
    🚀 빠른 시작

    1. 스코어링 엔진 실행

    $ cd engine
$ python3 scoring_engine.py PL

    2. 보고서 렌더링

    $ python3 report_renderer.py report/country/PL/PL_rpt_2026-06-18T1500.json

    3. 백엔드 서버 시작

    $ cd app/back
$ pip install fastapi uvicorn
$ python3 app.py

    4. 보고서 확인

    HTML 보고서: report/samples/PL_rpt_2026-06-18T1500.html
    API 문서: http://localhost:8000/docs
